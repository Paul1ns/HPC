Тестирование на изображении в градациях серого показало совпадение карт углов между CPU и GPU, что подтверждает корректность реализации. Ускорение достигается за счет того, что потоки одновременно вычисляют градиенты и меру Харриса для разных пикселей. Текстурная память автоматически подставляет значения для пикселей за краями изображения, поэтому не нужно писать дополнительные проверки границ.
