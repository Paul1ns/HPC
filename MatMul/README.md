## Платформа и оборудование
- **Платформа:** Google Colab  
- **Процессор (CPU):** виртуальный CPU Colab  
- **Графический процессор (GPU):** NVIDIA T4  

## GPU-реализация
Каждый поток GPU вычисляет один элемент результирующей матрицы, выполняя умножение строки одной матрицы на столбец другой.  
Потоки организованы в двумерную сетку, чтобы соответствовать структуре матрицы (строки и столбцы).

## Проведённые эксперименты
- Использованы матрицы размеров: `100`, `500`, `1000`, `1500` и `2000`.  
- Для каждого размера измерялось **время умножения на CPU и GPU**.  
- Тесты выполнены для трёх типов данных: `int`, `float`, `double`.  
- Время усреднено по **трём запускам** для более точной оценки.

## Графики и результаты
### Время выполнения умножения
- Время на CPU **быстро растёт** с увеличением размера матрицы.  
- GPU обеспечивает **значительное ускорение**, особенно для больших матриц.  
- Время на GPU стабильно **значительно меньше**, чем на CPU, для всех типов данных.  
- Время работы на типе `double` немного выше, чем на `float` и `int`, что связано с более высокой точностью.
### Ускорение GPU относительно CPU
- Ускорение GPU **растёт с размером матрицы**.  
- Для `int` и `float` достигается **максимальное ускорение**.  
- Для `double` ускорение меньше, что связано с особенностями вычислений с двойной точностью.
