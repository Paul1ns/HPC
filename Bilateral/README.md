## Платформа и оборудование
- **Платформа:** Google Colab  
- **Процессор (CPU):** виртуальный CPU Colab  
- **Графический процессор (GPU):** NVIDIA T4  

## GPU-реализация
Каждый поток обрабатывает один пиксель изображения, применяя билатеральный фильтр по окрестности 3×3. Используется texture memory для ускорения доступа к данным

## Проведённые эксперименты
- Запуски с различными значениями sigma_d и sigma_r
- Для каждого запуска измерялось **время выполнения на CPU и GPU**.  


## Результаты
В файле results.csv содержатся данные:
- var_cpu, var_gpu: оценивает степень сглаживания
- mse_cpu, mse_gpu: отличие от исходного изображения
- mse_cpu_gpu: разница между CPU и GPU
