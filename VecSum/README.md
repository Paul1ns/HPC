## Платформа и оборудование
- **Платформа:** Google Colab  
- **Процессор (CPU):** виртуальный CPU Colab  
- **Графический процессор (GPU):** NVIDIA T4  

## GPU-реализация
Каждый поток обрабатывает несколько элементов массива, после чего выполняется редукция внутри блока с использованием shared memory. Частичные суммы блоков копируются на CPU, где выполняется финальное суммирование.

## Проведённые эксперименты
- Использованы массивы размеров: `1000`, `10000`, `100000`, `500000`, `1000000` элементов.
- Для каждого размера измерялось **время умножения на CPU и GPU**.  
- Тесты выполнены для трёх типов данных: `int`, `float`, `double`.  
- Время усреднено по пяти запускам для более точной оценки.
